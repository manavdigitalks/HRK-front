# Design Beautiful UI

This is a code bundle for Design Beautiful UI. The original project is available at https://www.figma.com/design/ECXxGCcfijV5sOdi2yZHKh/Design-Beautiful-UI.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

## Tech Stack

- Next.js 15
- React 18
- TypeScript
- Tailwind CSS 4
- Radix UI Components
